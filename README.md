# homebrew-tap

Homebrew formulae for [grikwong](https://github.com/grikwong) projects.

## Usage

```bash
brew tap grikwong/tap
brew install crossagent
```

Or install directly:

```bash
brew install grikwong/tap/crossagent
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [crossagent](https://github.com/grikwong/crossagent) | Cross-model AI agent orchestrator with browser-based Web UI |
